# LLM Work Assistant (Web App)

PCに不慣れな方でも使いやすいよう、チャット形式UIのWebアプリを実装しました。

## 主な機能

- チャット形式でLLMに質問
- 右上「設定」で **各ユーザーのAPIトークン** を登録（ブラウザ保存）
- OpenAI API呼び出し（provider: openai）
- ナレッジベース（契約書ひな型・条文）を保存/削除
- ナレッジベースを参照した「契約書ドラフト作成」
- 入力ファイル + プロンプトCSVの一括分析（複数プロンプト実行）

## 起動

```bash
python -m app.webapp
```

ブラウザで `http://localhost:8000` を開いて利用します。

## 使い方（最短）

1. 右上「設定」から OpenAI APIトークンを保存
2. 左のナレッジベースに契約書の雛型条文を登録
3. チャット欄に要件を書いて「契約書ドラフト作成」を押す
4. 通常チャットで質問、または入力ファイル+CSVで一括分析

## 注意点

- APIトークンはブラウザの `localStorage` に保存されます。
- サーバー側は最小構成です。運用時は認証、監査ログ、暗号化、アクセス制御を追加してください。
